# 🌍 Global Dev Gigs

A modern job listing UI built with React that displays developer job openings in a clean, interactive card layout.

## 📌 Features

* 🧩 Dynamic job cards
* 🔖 Save/bookmark functionality (UI)
* 🔗 Apply button opens job link in new tab
* 🏷️ Job tags (Full-time, Junior, etc.)
* 💻 Clean and responsive UI

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript (ES6+)
* CSS
* Lucide Icons

## ⚙️ Installation & Setup

Clone the repo:

```bash
git clone https://github.com/your-username/global-dev-gigs.git
cd global-dev-gigs
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
